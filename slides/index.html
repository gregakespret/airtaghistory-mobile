The signature feature

Drag the slider.
Watch time run backward. ⚡

Scrub the time-travel control and every pin jumps to where it was at that instant, drawing the trail it took to get there. Tap ⚡ Live to snap back to the present.

Pure & testable

buildTimeline · positionsAt · trailFor — the interpolation logic is pure functions with Jest tests. The map screen just renders what they return.

The app
Expo · React Native
Thin, read-only. Renders maps, pins, and the time-travel timeline. Never sees your Apple ID.
Why it's read-only by design: all the risky Apple-data collection lives on the server behind a trust boundary. The app just needs a login token — so it stays simple, low-risk for the App Store, and impossible to leak credentials it never holds.
